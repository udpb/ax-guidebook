1.2현장 인터뷰 구조화 — 대표·실무자·공장장
가이드북 2.3의 인터뷰 구조화 패턴을 실제 현장에 적용한 결과다. 각 발언 아래 구조화 해석을 직접 확인하라.
대표 김정우 — 전체 사업 현황 (90분)6개 병목 전수 발견▶
"매출은 80억이 넘었는데, 남는 게 없다고 하셨는데 좀 더 구체적으로?" FACTPAIN
"작년에 82억 찍었는데, 영업이익이 한 3억 정도? 이익률이 4%도 안 돼요. 2년 전엔 매출 55억에 이익이 4억이었거든요. 매출은 50% 늘었는데 이익은 줄었어요."
구조화매출 55억→82억(+49%), 영업이익 4억→3억(−25%). 이익률 7.3%→3.7%. 매출 성장이 수익성을 갉아먹는 구조. → 복잡도 비용 (#2), 측정 부재 (#3)
"어디서 비용이 새는 것 같으세요?" CAUSEPAIN
"솔직히 정확히 모르겠어요. 쿠팡이 커지면서 수수료가 확 늘었고, 물류비도 올랐고... 근데 채널별로 정확히 얼마 남는지는 계산해본 적 없어요. 전체 매출에서 전체 비용 빼는 수준이에요."
구조화채널별 공헌이익 미측정. 전체 P&L만 있고 채널/SKU별 수익성 분석 부재. → 측정 부재 (#3), 데이터 파편화 (#1)
"SKU가 87개던데, 어떻게 이렇게 많아졌어요?" CAUSE
"처음엔 프로틴바 3종으로 시작했는데, 고객이 맛 다양화를 원하니까 늘리고, 쿠팡에서 번들 상품을 요구하니까 또 늘리고, 올리브영 전용 사이즈 만들고... 지금은 어떤 게 잘 팔리고 어떤 게 재고만 쌓이는지 한눈에 안 보여요."
구조화SKU 증가 원인이 전략적 판단이 아닌 채널별 요구 대응. SKU별 수익성 미측정 상태에서 관성적으로 증가. → 복잡도 비용 (#2)
"공장 쪽은 어떤 상황인가요?" FACTPAIN
"공장이 가장 골치예요. SKU가 87개니까 라인 전환이 하루에 3~4번이에요. 전환할 때마다 30분~1시간 세척하고 세팅하는데, 한 달이면 50시간은 되거든요. 소량 생산 SKU가 많아서 원재료를 소량 발주하게 되고, 단가도 올라가요."
구조화라인 전환 50h/월 = 인건비 약 150만원/월 + 기회비용(생산 가능 제품). 소량 발주 → 원재료 단가 상승. "매출은 느는데 남는 게 없다"의 숨은 원인. → 복잡도 비용 (#2)
"주문 관리는 어떻게 하세요?" WORKAROUND
"이 대리가 매일 아침 카페24, 쿠팡, 네이버, 올리브영 발주서 다 취합해서 엑셀로 정리해요. 3시간은 걸리죠. 그 엑셀을 공장에 보내면 공장장이 생산 계획을 짜는데, 또 수기로 해요."
구조화주문 취합 3h/일 × 22일 = 66h/월. 수기 생산 계획. 채널별 데이터 분리로 통합 뷰 없음. → 데이터 파편화 (#1), 느린 의사결정 (#4)
"재고 관리는요?" PAIN
"완제품 재고는 이 대리 엑셀이에요. 원재료 재고는 공장 김 주임이 별도 엑셀로 관리하고. 둘이 맞춰본 적은... 분기에 한 번 정도? 가끔 쿠팡에서 주문 터지면 재고가 없어서 품절 걸리고, 그러면 검색 순위 떨어지고..."
구조화완제품 재고(이 대리 엑셀) ↔ 원재료 재고(공장 김 주임 엑셀) = SSOT 부재. 품절 → 쿠팡 검색순위 하락 → 매출 손실. → 데이터 파편화 (#1), 데이터 비자산화 (#5)
이 대리 — 운영팀 주문/재고 담당반복 수작업 3.5h/일▶
"매일 아침 루틴이 어떻게 되세요?" FACTWORKAROUND
"8시 반에 출근하면 카페24 엑셀 뽑고, 쿠팡 Wing 뽑고, 네이버 커머스 뽑고, 올리브영 이메일 확인하고. 네 개 양식이 다 달라서 통일하는 데만 1시간 넘게 걸려요. 그 다음에 물류사 발주서 변환해서 보내고, 품절 처리하고... 점심 전에 겨우 끝나요."
구조화주문 취합+양식 통일+발주 변환+품절 처리 = 약 3.5h/일. 이 중 판단이 필요한 시간은 약 15분. 나머지 3시간 15분은 반복 수작업. → 데이터 파편화 (#1), 복잡도 비용 (#2)
"B2B 납품은 어떻게 관리하세요?" PAIN
"그건 박 과장이 따로 해요. 저는 잘 몰라요. 박 과장 PC에 있을 거예요. 가끔 B2B 물량이랑 D2C 물량이 겹쳐서 공장에서 뭘 먼저 생산해야 하는지 헷갈릴 때가 있어요. 그때는 대표님한테 물어봐요."
구조화B2B와 D2C 데이터 완전 분리. 생산 우선순위 판단이 대표에게 집중 → 의사결정 병목. → 데이터 파편화 (#1), 느린 의사결정 (#4)
공장장 최 부장 — 생산/원재료 담당긴급 발주 연 2,400만 추가 지출▶
"생산 계획은 어떻게 세우세요?" FACT
"이 대리가 엑셀 보내주면 그거 보고 저녁에 다음 날 생산 계획을 짜요. SKU가 많으니까 라인 전환이 문제예요. 프로틴바 → 그래놀라 → 쉐이크 이렇게 바꿀 때마다 세척하고 설정 바꾸는 데 40분은 걸려요. 하루에 3~4번 바꾸니까 2시간 넘게 허비해요."
구조화라인 전환 40분 × 3.5회/일 × 22일 = 약 51h/월. 시간당 인건비 3만원 기준 153만원/월 + 생산량 감소 기회비용 추정 월 2,000만원.
"원재료 발주는 어떻게?" FACTPAIN
"한 달에 한 번 몰아서 발주하는데, 갑자기 매출 터지면 긴급 발주를 넣어야 해요. 긴급 발주는 단가가 10~15% 비싸요. 올해만 긴급 발주가 8번이었어요."
구조화긴급 발주 8회/년 × 평균 300만원 추가 비용 = 연 2,400만원 추가 지출. 수요 예측 부재가 원인. → 복잡도 비용 (#2), 측정 부재 (#3)
